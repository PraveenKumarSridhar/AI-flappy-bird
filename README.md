### Introduction
<hr/>

Build Flappy bird game and teach AI to build it.

### Packages required
<hr/>

```
pip install pygame
pip install neat-python
```

### Steps to run
<hr/>

Just run the `flappy_bird_game.py` and watch as the AI train itself to play the game file as follows:

```
python flappy_bird_game.py 
```
